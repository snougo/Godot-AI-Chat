# multi_replace 工具使用说明

## 适用场景
当需要对脚本中的多处相同文本进行同时替换时使用。基于精确字符匹配定位，无需关心行号。

## 操作类型

每个操作只有两个字段：`search` 和 `content`。

| 字段 | 说明 |
|------|------|
| `search` | 要搜索的精确文本（支持 `\\n` 表示换行） |
| `content` | 替换后的文本（支持 `\\n` 表示换行） |

## 核心规则

1. **所有搜索均基于修改前的原始文件**，操作之间互不影响
2. 每个 search 会匹配文件中**所有不重叠**的出现位置，全部替换
3. **不同操作之间的匹配区域不能重叠**，否则会报错
4. 搜索不到的操**作会跳过并报告警告**，不影响其他操作执行
5. search 必须完全匹配（包括空格、缩进、换行）

## 示例

同时重命名函数 + 修改继承：

```json
{
  "operations": [
	{
	  "search": "func old_name():\\n    return 42",
	  "content": "func new_name():\\n    return 100"
	},
	{
	  "search": "extends Node",
	  "content": "extends Node2D"
	}
  ]
}
