# Image Gen

## 概览
本技能允许你通过调用本地运行的 **Draw Things** app 的 HTTP API 来生成图像。

## 工作流

1. 将任务描述中用户提供的提示词进行细化丰富，并反推负面提示词
2. 调用 `draw_things_generate_image` 并等待结果
3. 无论成功与否，都使用 `report_task_result` 进行回报

## 最佳实践

1. **提示词越详细越好** — 描述主体、环境、风格、光照、色彩等
2. **合理使用负面提示词** — 排除不需要的元素
3. **种子复用** — 用固定种子（非 -1）可复现同一张图微调
4. 通常无需自定义工具可选参数，使用默认值即可，除非用户明确要求使用特殊的参数值。

## 注意事项

- 生成时间取决于模型大小、步数、分辨率以及本地电脑的性能，有可能因为生成时间过长导致工具调用超时。
- 如果 Draw Things 未运行，工具会返回连接失败的错误信息
