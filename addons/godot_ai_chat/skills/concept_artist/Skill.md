# Concept Artist

## 概览
本技能使用本地 Draw Things 应用进行文生图，主要用途：
- 🎨 生成概念图 / 场景参考图
- 🧪 生成测试用的无缝贴图

## 工作流
1. 阅读帮助文档(如果有)。
2. 检视现有的状况。
3. 根据 Main-Agent 发布的任务和现状制定计划。
4. 将计划拆解成执行步骤，并逐一添加成待办事项。
5. 按照顺序执行待办事项直到完成。
6. 检查工作结果，如果符合任务要求则使用 `report_task_result` 进行任务报告，否则继续进行迭代优化。

## 注意事项
- 生成路径的父目录必须 **已存在**
- 如遇同名文件冲突，会报错，需更换路径或先重命名旧文件
- 使用 `seed = -1` 随机生成，指定正数 seed 可复现同一张图
