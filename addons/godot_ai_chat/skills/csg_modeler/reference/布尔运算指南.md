# 布尔运算指南

布尔运算是 CSG 建模的核心机制。通过在 `CSGCombiner3D` 下组织子节点，并为每个子节点设置不同的 `operation` 值，可以实现复杂的几何组合。

| 运算 | 值 | 效果 | 应用场景 |
|------|-----|------|---------|
| **UNION** (并集) | `0` | 合并两个形状 | 拼接基本体构建复杂外形 |
| **INTERSECTION** (交集) | `1` | 保留重叠部分 | 裁剪出特定交叉区域 |
| **SUBTRACTION** (差集) | `2` | 从父形状中减去子形状 | 挖孔、开槽、雕刻细节 |

**典型层级结构示例：**

```
CSGCombiner3D (operation=0, UNION)
├── CSGBox3D (operation=0, UNION)      ← 主体
│   └── CSGSphere3D (operation=2, SUBTRACTION)  ← 在盒体上挖出球面凹槽
└── CSGCylinder3D (operation=0, UNION) ← 附加圆柱
```

**关键规则**：
- `CSGCombiner3D` 内部的所有子节点根据其 `operation` 属性相互进行布尔运算
- 子节点可以嵌套，即一个 `CSGCombiner3D` 可以包含另一个 `CSGCombiner3D`
- 每个 CSG 节点通过 `position` / `rotation` / `scale` 控制其空间变换
