# File Organizer

## 概览
本技能用于整理 Godot 项目的文件结构，包括创建分类文件夹、移动文件到对应目录、以及统一文件命名规范。

> **注意**：操作前务必先获得用户确认，避免误操作导致文件丢失。

## 工作流

1. **分析需求**：与用户沟通，明确文件整理的目标和规则
2. **扫描项目结构**：使用 `manage_folder` 查看目标文件夹的当前结构
3. **制定整理计划**：规划分类目录结构和文件移动/重命名方案，提交用户确认
4. **创建分类目录**：使用 `manage_folder` 按计划创建目标文件夹
5. **移动文件**：使用 `move_file` 将文件移动到对应的分类目录
6. **重命名文件**：使用 `rename_file` 按命名规范统一文件名称
7. **确认结果**：再次使用 `manage_folder` 展示整理后的结构

## 注意事项

- `move_file` **不支持**移动文件夹，仅支持移动单个文件
- `move_file` 不允许在移动时改变文件名（如需改名请先用 `rename_file`，再移动）
- `rename_file` 仅允许修改文件名，不允许改变文件所在目录（如需移动请用 `move_file`）
- `move_file` 的 `target_path` 可以是一个已存在的文件夹路径，文件会被移入该文件夹内；也可以是一个完整的目标文件路径
- 所有路径必须以 `res://` 开头
- 执行写操作（移动/重命名/创建文件夹）后，编辑器文件系统会自动刷新
